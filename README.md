> 这是一个长期支持的微信小程序博客(JavanorthApp-LTS)

## 本地运行
```shell script
git clone https://github.com/hirCodd/JavaNorthMiniApp.git
cd JavaNorthMiniApp
npm install
npm dev:weapp
```
## 本地展示
```
1.打开本地微信小程序工具
2.打开dist目录运行即可
```
